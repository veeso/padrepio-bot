## Get started

```sh
cargo install padrepio-discord-bot
export CHATGPT_API_KEY=xxxx
export DISCORD_TOKEN=xxxx
export SERVER_GUID=xxxx
export RUST_LOG=info
padrepio-discord-bot
```

## Available commands

- `/padrepio <query>` ask padrepio
